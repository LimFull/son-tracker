# Son Tracker

손흥민 선수의 토트넘 경기 일정과 결과를 추적하는 웹 애플리케이션입니다.

## 주요 기능
- 토트넘의 경기 일정 및 결과 확인
- 실시간 경기 스코어 업데이트
- 경기 시작 전 알림 서비스 (준비 중)
